# GanaderíaApp

Sistema de Gestión Ganadera con Inteligencia Artificial.

## Tecnologías
- **Frontend**: HTML + CSS + JavaScript vanilla (un solo archivo)
- **Base de datos**: Firebase Firestore
- **Autenticación**: Firebase Auth con Google
- **Hosting**: GitHub Pages

## Acceso
Abrir: https://[tu-usuario].github.io/ganaderia-app

## Seguridad
Cada usuario solo accede a sus propios datos mediante Firebase Security Rules.
